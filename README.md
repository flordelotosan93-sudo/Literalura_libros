Literalura - Catálogo de Libros

Literalura es una aplicación de consola desarrollada en Java que permite buscar y gestionar información sobre libros y autores. La aplicación consume datos desde la API pública de Gutendex, los procesa en formato JSON y los almacena en una base de datos relacional PostgreSQL para su posterior consulta.

Este proyecto fue desarrollado como parte del programa Oracle Next Education (ONE), con el objetivo de aplicar y reforzar conocimientos en Java, Spring Boot y gestión de bases de datos.

A través de Literalura es posible:

Buscar libros utilizando la API de Gutendex.

Guardar la información obtenida en una base de datos.

Consultar libros registrados en el sistema.

Obtener información sobre autores y sus obras.

Realizar búsquedas de autores que estaban vivos en un año determinado.

Filtrar libros por idioma.

El proyecto integra tecnologías como Spring Boot, JPA/Hibernate, PostgreSQL y el consumo de APIs REST, permitiendo trabajar con procesamiento de datos, persistencia y consultas estructuradas.
